# Project README

## Описание проекта

Этот проект был создан как экспериментальный/дизайн-проект с элементами юмора и нестандартной визуальной подачи.

## Работа над проектом

В процессе разработки я активно использовал помощь AI-ассистента (ChatGPT) для:

- генерации идей и концепций
- помощи в написании текстов
- разработки UI-решений
- решения технических задач (HTML / CSS / JS)

## Роль AI

AI использовался как вспомогательный инструмент для ускорения разработки и поиска решений, при этом все финальные решения и реализация контролировались мной.

## Технологии

- HTML
- CSS
- JavaScript (если есть)
- Figma (на этапе дизайна)

## Примечание

Проект носит экспериментальный характер и создан в рамках обучения и практики.
